### Python nb0-unpack
Requires Python 3.6+  

#### Usage
```
usage: nb0-unpack.py [-h] [-d] [-i] nb0file [outdir]

positional arguments:
  nb0file     .nb0 file
  outdir      output directory to extract files

options:
  -h, --help  show this help message and exit
  -d          enable debug output
  -i          print information of file and exit
```
